Demo 2: Welch's T-Test (Continuous Metrics)
============================================================

  Experiment: RAG top-k=5 vs top-k=3
  Metric: Mean retrieval similarity score (continuous)

  Variant             N     Mean      Std
  ———————————— ———————— ———————— ————————
  Control          5000   0.7208   0.1495
  Treatment        5000   0.7586   0.1414

  Mean difference: +0.0378
  Relative lift: +5.24%
  T-statistic: 12.9800
  P-value: 0.000000
  95% CI: [+0.0321, +0.0435]
  Significant (α=0.05): Yes

  Key insight: For continuous metrics (latency, similarity scores),
  use t-test instead of the two-proportion z-test.

Demo 3: Bonferroni Correction (Multiple Comparisons)
============================================================

  Testing 3 metrics simultaneously:

  Metric                    P-value     α=0.05  Significant?
  —————————————————————— —————————— —————————— —————————————
  conversion_rate          0.019112      0.050         Yes ✓
  click_through_rate       0.012441      0.050         Yes ✓
  engagement_score         0.533376      0.050            No

  After Bonferroni correction (α/3 = 0.0167):

  Metric                    P-value     Adj. α   Original  Corrected
  —————————————————————— —————————— —————————— —————————— ——————————
  conversion_rate          0.019112     0.0167        Yes       No ✗
  click_through_rate       0.012441     0.0167        Yes      Yes ✓
  engagement_score         0.533376     0.0167         No       No ✗

  ⚠ 1 metric(s) lost significance after correction
  This prevents false discoveries from testing multiple metrics

Demo 4: Confidence Interval Decision Framework
============================================================

  Scenario                        P-value                 95% CI Decision
  —————————————————————————————— ———————— —————————————————————— ————————————————————
  Strong positive effect           0.0000     [+0.0124, +0.0264] SHIP TREATMENT
  Weak positive (underpowered)     0.4338     [-0.0210, +0.0090] EXTEND EXPERIMENT
  No effect (null result)          0.1156     [-0.0121, +0.0013] NO DIFFERENCE
  Negative effect                  0.0009     [-0.0178, -0.0046] KEEP CONTROL

  Decision rules:
    CI entirely > 0 & significant → SHIP TREATMENT
    CI entirely < 0 & significant → KEEP CONTROL
    CI wide (spans 0, range > 2%) → EXTEND EXPERIMENT
    CI narrow around 0            → NO DIFFERENCE
